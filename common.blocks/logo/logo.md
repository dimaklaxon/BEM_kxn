# Logo

OJSC Alfa-Bank logo.

# Usage

``` js
{
    block : 'logo'
}
```

## Block mods

- light background (white)

``` js
{
    block : 'logo',
    mods : { theme  : 'alfa-on-white' }
}
```

- colorful background (color)

``` js
{
    block : 'logo',
    mods : { theme  : 'alfa-on-color' }
}
```

### _size

- small (s)

``` js
{
    block : 'logo',
    mods : { size : 's' }
}
```

- medium (m)

``` js
{
    block : 'logo',
    mods : { size : 'm' }
}
```

- large (l)

``` js
{
    block : 'logo',
    mods : { size : 'l' }
}
```

- x large (xl)

``` js
{
    block : 'logo',
    mods : { size : 'xl' }
}
```

### _view

- logo with text (full)

``` js
{
    block : 'logo',
    mods : { view : 'full' }
}
```

- only letter (icon)

``` js
{
    block : 'logo',
    mods : { view : 'icon' }
}
```
